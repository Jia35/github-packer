GitHub 認證設定

設定 Personal Access Token (PAT) 以支援下載私有儲存庫的內容。

💡 設定提示

  • 必要權限 (Fine-grained)：Repository access 選擇 All repositories，Permissions 僅需勾選 Contents (Read-only) 即可。
  • 隱私說明：此 Token 僅會儲存在您本地瀏覽器的 chrome.storage 中，不會傳送至任何第三方伺服器。
  • GitHub 官方文件：如何建立 GitHub Personal Access Token？
